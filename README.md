# TaskFlow

## Descrição

O TaskFlow é um sistema simples de gerenciamento de tarefas desenvolvido para auxiliar equipes ágeis no controle de atividades.

## Objetivo

Permitir o cadastro, conclusão e exclusão de tarefas de forma simples e intuitiva.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Jest
- GitHub Actions

## Metodologia Ágil

Foi utilizada a metodologia Kanban para organização das tarefas do projeto.

## Funcionalidades

- Adicionar tarefas
- Concluir tarefas
- Excluir tarefas
- Prioridade de tarefas

## Mudança de Escopo

Durante o desenvolvimento, foi adicionada a funcionalidade de prioridade das tarefas para melhorar a organização das atividades.

## Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html`
